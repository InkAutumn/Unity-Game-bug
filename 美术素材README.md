### 文件格式要求
- **手部/角色**: PNG-24（透明背景）
- **背景图**: JPG 90%质量 或 PNG
- **UI元素**: PNG（透明背景）
- **图标**: PNG（透明背景）

### 色彩规范
```
UI元素：
- 按钮背景 #E8D4B8
- 面板浅色 #F0F0F0 (240, 240, 240)
- 半透明遮罩 rgba(0, 0, 0, 0.7)
```

### 命名规范
```
格式：{类型}_{名称}_{状态}.{扩展名}

示例：
hands_grandma_normal.png
bg_kitchen.jpg
button_normal.png
icon_menu.png
dumpling_perfect.png
```
